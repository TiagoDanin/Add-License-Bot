About

Generate LICENSE file from package.json by Tiago Danin.

Add License Bot

Summary

If your repository does not have a file with name:

  • LICENSE
  • LICENSE.md
  • license
  • license.md

A license file will be generated from package.json.

Permission

  • Read access to code.
  • Read and write access to single file ("LICENSE").

Notification

The bot will mention you in a comment.

Public App

  • github.com/apps/Add-License

Setup

# Install dependencies
npm install

# Run the bot
npm start

Tests

To run the test suite, first install the dependencies, then run test:

# NPM
npm test
# Or Using Yarn
yarn test
